# Коробочные оповещения

GMONIT поставляет готовый пакет преднастроенных правил оповещения, покрывающих APM, браузерный мониторинг, инфраструктуру и базы данных. Десятки правил, которые обычно заводятся вручную для каждой метрики, уже собраны в компактный набор и работают из коробки — пакет нужно только включить.

Для тонкой настройки доступен интерфейс переопределений: пороги срабатывания и другие параметры правила меняются для отдельной сущности (приложения, хоста, метрики), не трогая весь пакет.

Дополнительно поддерживается эскалация по матрице RACI: пока нарушение держится, на оповещении проставляется уровень `L1` → `L2` → `L3`, и по этой метке разные уровни можно направить разным получателям. Эскалация по RACI пока работает только для правил из коробочного пакета; её профили и маршрутизация настраиваются в интерфейсе GMONIT.
