# Коробочные оповещения

GMONIT поставляет готовый пакет преднастроенных правил оповещения, покрывающих APM, браузерный мониторинг, инфраструктуру и базы данных. Десятки правил, которые обычно заводятся вручную для каждой метрики, уже собраны в компактный набор и работают из коробки — пакет нужно только включить.

Для тонкой настройки доступен интерфейс переопределений: пороги срабатывания и другие параметры правила меняются для отдельной сущности (приложения, хоста, метрики), не трогая весь пакет.

Дополнительно поддерживается эскалация по матрице RACI: пока нарушение держится, на оповещении проставляется уровень `L1` → `L2` → `L3`, и по этой метке стандартными политиками уведомлений Grafana можно направить разные уровни разным получателям. Эскалация по RACI пока работает только для правил из коробочного пакета; её параметры настраиваются в том же интерфейсе, что и переопределения порогов.

## Возможности

- Готовый пакет правил для APM, браузерного мониторинга, инфраструктуры и баз данных: десятки правил сведены к компактному набору
- [Переопределение параметров правила для отдельной сущности](/modules/alert-pack/guide.md#переопределить-параметры-оповещения-для-сущности): пороги, область применения, включение и выключение
- [Эскалация оповещения по уровням ответственности (RACI)](/modules/alert-pack/guide.md#просмотреть-активные-оповещения): уровень `L1` → `L2` → `L3` повышается, пока нарушение держится. Настраивается там же, где переопределения порогов
- [Ограничение переопределений по командам Grafana](/modules/alert-pack/setup.md#ограничение-переопределений-по-командам): пользователь работает только с переопределениями своей команды
- [Включение пакета и настройка интервала оценки](/modules/alert-pack/setup.md#включение-пакета-оповещений-и-интервал-оценки)
