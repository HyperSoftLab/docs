# RACI-алерты

RACI-алерты — это готовый пакет преднастроенных правил оповещения с эскалацией по уровням ответственности. Вместо ручной настройки десятков правил на каждую метрику пакет покрывает APM, браузерный мониторинг, инфраструктуру и базы данных. По мере того как нарушение сохраняется во времени, пакет проставляет на оповещении метку уровня эскалации — а по этой метке уже легко настроить, кому и когда уходит уведомление, через стандартные политики уведомлений Grafana.

## Как это работает

Пакет содержит небольшой набор обобщённых правил, каждое из которых покрывает целый класс метрик. Когда правило срабатывает, пакет проставляет на оповещении метку уровня эскалации `gmonit_raci.level`: `L1` — сразу, `L2` и `L3` — по мере того, как нарушение продолжает держаться. Сам пакет уведомления никому не рассылает — он лишь размечает оповещения уровнями. Маршрутизацию по этим меткам (`L1` → дежурной команде, `L2` → старшему, `L3` → руководству) вы настраиваете через стандартные политики уведомлений Grafana. Метка снимает с вас ручную работу: не нужно заводить отдельные правила под каждый уровень — достаточно одного условия на метку в политике.

Вся логика расчёта уровней выполняется на стороне базы данных при оценке правила — отдельных сервисов и фоновых процессов не требуется. Параметры конкретного правила (пороги, важность, задержки эскалации) можно переопределить для отдельной сущности, не затрагивая весь пакет.
