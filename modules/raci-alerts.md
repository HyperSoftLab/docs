# RACI-алерты

RACI-алерты — это готовый пакет преднастроенных правил оповещения с эскалацией по уровням ответственности. Вместо ручной настройки десятков правил на каждую метрику пакет покрывает APM, браузерный мониторинг, инфраструктуру и базы данных, а при затяжном нарушении автоматически повышает уровень оповещения от исполнителя к руководству.

## Как это работает

Пакет содержит небольшой набор обобщённых правил, каждое из которых покрывает целый класс метрик. Когда нарушение сохраняется во времени, оповещение эскалируется по уровням RACI: чем дольше длится проблема, тем выше уровень, на который она поднимается. Каждому уровню соответствует метка, по которой стандартная маршрутизация уведомлений Grafana доставляет оповещение нужной команде.

Вся логика эскалации вычисляется на стороне базы данных при оценке правила — отдельных сервисов и фоновых процессов не требуется. Параметры конкретного правила (пороги, важность, задержки эскалации) можно переопределить для отдельной сущности, не затрагивая весь пакет.
