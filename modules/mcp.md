# MCP

> ⚠️ **Экспериментальная функция.** Не рекомендуется для production-окружений.

MCP-сервер коллектора, через который AI-агенты (Claude, Codex и подобные) ходят в данные GMonit на естественном языке. Сценарий — встроить GMonit в инструментальный пайплайн разработчиков, которые уже работают с агентами и не идут в UI ради быстрых вопросов.

## Как это работает

Коллектор поднимает MCP-эндпоинт по протоколу Streamable HTTP. Клиент подключается к нему по Bearer-токену и получает набор инструментов поверх ClickHouse: Apdex и здоровье APM-приложений, схема таблиц, состояние pipeline, опционально — произвольный SQL. Какой инструмент дёрнуть, агент решает сам по формулировке вопроса.
