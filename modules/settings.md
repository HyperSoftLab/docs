# Настройки

Настройки позволяют управлять параметрами работы GMonit через интерфейс Grafana: сроки хранения данных, правила обработки транзакций и конфигурация агентов.

## Как это работает

Параметры системы настраиваются через страницы конфигурации в интерфейсе GMonit. Изменения применяются к коллектору и хранилищу данных без необходимости перезапуска сервисов. Для некоторых настроек требуется предварительное включение через переменные окружения коллектора.

## Возможности

- [Настройка сроков хранения данных](/modules/settings/guide.md#настроить-сроки-хранения-данных) — управление TTL для каждой категории данных (APM метрики, события, ошибки, трассировки, RUM, инфраструктура)

## Планируемые возможности

- Правила переименования и исключения транзакций — управление именами транзакций через интерфейс (замена, группировка, исключение)
- Удалённая конфигурация агента — управление настройками APM-агентов через интерфейс без доступа к хосту (исключения ошибок, статус-коды, кастомная инструментация)
- Обнаружение классов и методов — просмотр классов и методов из профилирования и трейсов для настройки кастомной инструментации
